* BA-Sorterキャラ名日本語変換 *

BA-Sorterで表示される英語のブルアカキャラ名を日本語に変換して表示します

BA-Sorter 2025-02-07 Update Ver のキャラ一覧に対応しています

キャラクター日本語名は「ブルーアーカイブ（ブルアカ）攻略 Wiki」を参考にしています
https://bluearchive.wikiru.jp/?プロフィール一覧


===使い方===

① Chromeで chrome://extensions/ を開く

②「デベロッパーモード」ON

③「パッケージ化されていない拡張機能を読み込む」から本拡張機能のフォルダを指定

④ BA-Sorterサイトにアクセス → 表示が自動的に日本語化される
